# scRNA-seq Pipeline: Normalization & Feature Selection Stage

## Overview

This stage performs normalization, feature selection, dimensionality reduction, and technical covariate correction on QC-filtered single-cell expression matrices.

Inputs to this stage are QC-filtered singlet SingleCellExperiment objects generated by the preprocessing pipeline.

Outputs are downstream-ready SCE objects containing:

- normalized expression (logcounts)
- highly variable gene annotations
- PCA embeddings
- PCA embeddings after technical covariate regression

This stage is implemented using Bioconductor SingleCellExperiment workflows and supports parallel execution for both:

- Cell Ranger outputs
- STARsolo outputs

---

## Inputs

Input objects originate from the QC stage:

analysis/objects/<dataset>_singlets_sce.rds

Examples:

analysis/objects/pbmc1k_cellranger_singlets_sce.rds
analysis/objects/pbmc1k_starsolo_singlets_sce.rds

These objects contain:

assays:
    counts

colData:
    barcode
    total_counts
    detected_genes
    mito_counts
    pct_mito
    scDblFinder metadata

---

## Pipeline Steps

The downstream stage consists of the following modules:

scripts/downstream/

10_normalize_hvg.R  
20_run_pca.R  
21_assess_pc_covariates.R  
22_regress_covariates_and_rerun_pca.R  
23_merge_pca_variants.R  
24_assess_any_reduceddim_covariates.R  

---

## Step 1 — Normalize Counts

Script:

scripts/downstream/10_normalize_hvg.R

Normalization is performed using:

scuttle::logNormCounts()

This performs:

- library size normalization
- log transformation

Resulting assay:

logcounts

---

## Step 2 — Identify Highly Variable Genes

Highly variable genes are identified using:

scran::modelGeneVar()

Genes are ranked by biological variance.

The top 2000 HVGs are selected.

Annotations added to rowData:

hvg_bio  
hvg_total  
hvg_tech  
is_hvg  

---

## Step 3 — Initial PCA

Script:

scripts/downstream/20_run_pca.R

Principal component analysis is performed using:

BiocSingular::runPCA()

PCA is run on:

logcounts[HVGs]

with:

centering = TRUE  
scaling = TRUE  

Result stored in:

reducedDim(sce, "PCA")

---

## Step 4 — Assess Technical Covariate Influence

Script:

scripts/downstream/21_assess_pc_covariates.R

Correlations between PCs and technical covariates are computed.

Covariates tested:

total_counts  
detected_genes  
pct_mito  

Output tables:

analysis/qc/*_pc_covariates.tsv

These quantify technical influence on PCA embeddings.

---

## Step 5 — Regress Technical Covariates

Script:

scripts/downstream/22_regress_covariates_and_rerun_pca.R

Gene expression is regressed against:

log_total_counts  
pct_mito  

Residual expression values are used to compute a corrected PCA.

Result stored as:

reducedDim(sce, "PCA_regressed")

---

## Step 6 — Merge PCA Representations

Script:

scripts/downstream/23_merge_pca_variants.R

This merges:

PCA  
PCA_regressed  

into a single downstream-ready object.

---

## Step 7 — Validate Regression

Script:

scripts/downstream/24_assess_any_reduceddim_covariates.R

Technical covariate influence is reassessed for both:

PCA  
PCA_regressed  

Regression is considered successful when technical correlations are reduced.

---

## Final Outputs

Final downstream-ready objects:

analysis/objects/pbmc1k_cellranger_downstream_ready_sce.rds  
analysis/objects/pbmc1k_starsolo_downstream_ready_sce.rds  

These contain:

assays:
    counts
    logcounts

rowData:
    HVG annotations

reducedDims:
    PCA
    PCA_regressed

## Design Rationale

This workflow uses Bioconductor-native tooling rather than Seurat to maintain compatibility with:

SingleCellExperiment  
BiocSingular  
scran  
scuttle  

Advantages include:

- pipeline transparency
- script modularity
- backend independence
- reproducibility
